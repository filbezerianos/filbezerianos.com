The Lab


I build tools and products that solve problems I care about. Some drawn from patterns I have seen in enterprise product work, others from curiosity about what’s possible with emerging technology.

Product Work Focus Tool

An AI-powered tool that analyses your product vision and strategy to prioritise features by impact. It automatically evaluates new features, calculating their overall impact and priority, saving product teams time and effort.

AI-Powered Property Photo Analyser

Property Visual Finder uses a fine-tuned vision AI model to analyse property photos and identify specific features, enabling advanced, data-driven property ranking, and searches based on visual characteristics.

Cloud Migration Strategy AI Tool

An AI-powered tool designed to determine the optimal Cloud migration strategy for applications. It analyses application details and aligns them with the organisation’s Cloud goals to deliver tailored recommendations.

Chat Agents Duo

A personal project where two AI chat agents engage in real-time discussions and debates on various topics. Users can customize the tone, style and LLM model of each agent to make things even more engaging.

Relationships Mapping and Visualisation Tool

A tool that leverages AI to transform unstructured interview data into a visual overview diagram, expediting the analysis of structures and processes and enabling the identification of inefficiencies.

Dad Jokes Chatty

PunGPT is personal project built for fun and powered by a lightweight LLaMA model, it delivers puns and dad jokes on demand, and sometimes when you least expect them. All wrapped in a clean, familiar chat UI.
